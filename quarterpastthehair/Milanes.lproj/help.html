FuzzyClock help


Features

At the moment the only feature is that FuzzyClock displays the current time in a "fuzzy" style in the menu bar. FuzzyClock has now no dock icon and no menu bar. Because of this a click to the time string shows a menu where you can find some basic functions. Also the time string got a tool tip with the current date/time. Move your mouse over the fuzzy time and find out ...

The format of "Localizable.strings" has changed and is now more flexible to place the hourString on different positions.

Currently following localizations are available:

  • Dutch
  • English
  • U.S. English
  • French
  • German
    • K&oum;lsch
    • Süddeutsch
  • Japanese
  • Spanish

All standard languages are available through the system control panel "Languages". Other localizations are available through FuzzyClock Preferences.
